# secit-api

## Descrição
REST API proposta para o sistema SECIT 2021 (Semana de Ciência e Tencnologia do IFRN Campus Natal - Zona Norte).

## Stack
- [Node](https://nodejs.org/en/)
- [Express](https://expressjs.com/pt-br/)
- [Mongoose](https://mongoosejs.com/)

## API Documentation
- Swagger
- Insomnia Exports
